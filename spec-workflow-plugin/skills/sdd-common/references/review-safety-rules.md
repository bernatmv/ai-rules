# Review Safety Rules (Shared)

Common safety rules for all review skills. Individual review skills may add
skill-specific rules alongside these.

## Contents

- [Rules](#rules)

## Rules

- Never modify source documents during review — review is read-only
- Always use structured output for review results
- Always present findings to user before any approval step
- Always complete all review dimensions before generating final scores
- Tier 1 script results are authoritative — do not override via AI judgment
