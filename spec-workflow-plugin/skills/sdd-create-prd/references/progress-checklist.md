# PRD Progress Checklist

Copy this checklist and track your progress:

```
PRD Progress:
- [ ] Step 0: Pre-flight validation
- [ ] Step 0.1: Load context
- [ ] Step 0.2: Establish session mode
- [ ] Step 0.3: Update mode (if targeted edit)
- [ ] Step 1: Problem exploration + write session state (--step 1)
- [ ] Step 2: Goals and success criteria + write session state (--step 2)
- [ ] Step 3: Scope decision + write session state (--step 3)
- [ ] Readiness Check 1: validate-readiness.py --gate pre-requirements --session-file
- [ ] Step 4: Requirements drafting + write session state (--step 4)
- [ ] Step 5: Stress testing + write session state (--step 5)
- [ ] Readiness Check 2: validate-readiness.py --gate pre-generation --session-file
- [ ] Step 6: Document generation → {prd-name} + delete session state
- [ ] Step 6.1: validate-prd.py {prd-name} (max 2 fix cycles)
- [ ] Step 6.2: Register {prd-name} in discovery manifest
- [ ] Step 7: Review and Approval for {prd-name}
- [ ] Step 8: Post-draft — update artifact status to approved
```
