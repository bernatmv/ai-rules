# AI video plugin skills

Skills bundled from several upstream repos (see [../README.md](../README.md) for attribution and licenses):

- `ai-video-storyboard` — multi-shot shot lists with visually consistent per-segment prompts ([aicontentskills/ai-video-storyboard-skill](https://github.com/aicontentskills/ai-video-storyboard-skill))
- `ai-video-prompt-enhancer` — turn a rough idea into one cinematic single-clip prompt ([aicontentskills/ai-video-prompt-enhancer](https://github.com/aicontentskills/ai-video-prompt-enhancer))
- `tiktok-reel-hook-generator` — scroll-stopping 1–3s visual hooks for short-form vertical video ([aicontentskills/tiktok-reel-hook-generator](https://github.com/aicontentskills/tiktok-reel-hook-generator))
- `video-prompting` — model-specific prompts (Seedance 2.0, Ovi, Sora, Veo 3, Wan 2.2, LTX-2/2.3) + character-sheet prompts ([Square-Zero-Labs/video-prompting-skill](https://github.com/Square-Zero-Labs/video-prompting-skill))
- `visual-video` — AI-director/editor prompting for Seedance, Kling, Veo, Runway, Luma, Pika, Sora ([smixs/visual-skills](https://github.com/smixs/visual-skills), upstream `video`)
- `visual-image` — image prompting for Nano Banana (NBP/NB2) and GPT Image 2 ([smixs/visual-skills](https://github.com/smixs/visual-skills), upstream `image`)
- `character-design-sheet` — character consistency via reference sheets and LoRA techniques ([inference-sh/skills](https://github.com/inference-sh/skills))

See [README.md](../README.md).
